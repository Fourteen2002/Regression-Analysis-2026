# 课堂汇报准备报告

## 我为什么选 Class 封装

我选择了 CustomOLS 类，因为 Class 封装可以把模型参数、协方差矩阵、残差和拟合结果都保存到同一个实例里。这样在处理北美和欧洲两个市场时，两个模型实例互不干扰，代码更清晰、可维护性更高。

## OOP 对比过程式的优势

1. 数据封装：每个模型实例保存自己的状态。
2. 多实例隔离：不同市场之间不会共用参数。
3. 统一接口：CustomOLS 和 sklearn 都支持 fit/predict/score，符合鸭子类型设计。

- 北美市场 F 检验结果：F=40840.7606, p-value=0.000000。
  ✅ 结论：北美市场广告投放显著影响销售。

  ✅ 结论：欧洲市场广告投放显著影响销售。
